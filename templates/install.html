Usar como app no celular

Adicione à tela inicial e abra em tela cheia, igual a um app instalado da loja — mas sem precisar de loja.

Tudo certo! Você já está usando o app no modo instalado.
{# Android / Chrome #}

No Android (Chrome)

1
Abra este site no Chrome

Se chegou aqui por outro navegador, abra o mesmo endereço no Chrome.

2
Toque no menu ⋮

É o ícone de três pontinhos no canto superior direito.

3
Toque em "Instalar app" ou "Adicionar à tela inicial"

Confirme. O ícone do app aparece junto com seus outros apps.

{# iPhone / Safari #}

No iPhone / iPad (Safari)

1
Abra este site no Safari

No iPhone só funciona pelo Safari (não pelo Chrome).

2
Toque no botão Compartilhar

É o quadrado com uma seta para cima, na barra de baixo (ou de cima).

3
Toque em "Adicionar à Tela de Início"

Role um pouco se não achar. Depois confirme em "Adicionar". Pronto!

Depois de instalado, é só abrir pelo ícone na tela inicial — ele abre em tela cheia, sem a barra do navegador. Funciona offline para abrir, mas precisa de internet para carregar seus treinos.
