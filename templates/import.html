Como exportar do Strava

  1. Entre no Strava no navegador (não funciona pelo app).
  2. Vá em Configurações → Minha conta → Baixar ou excluir sua conta.
  3. Clique em "Obter início" em Baixar solicitação (opcional).
  4. Você receberá um e-mail (pode levar algumas horas) com o link para um zip.
  5. Dentro do zip, pegue o arquivo activities.csv e suba aqui.

Mapeamento de esportes: Run/Trail Run → corrida · Swim → natação · WeightTraining/Workout/CrossFit → musculação · resto → outro. Quando o CSV não tem calorias para uma atividade, eu estimo via MET × peso × tempo usando o peso do atleta ativo.
